## Kawaii theme plugin

- This plugin's a whole re-write of Norcalli's plugin.
- It should be used along with [Kawaii](https://github.com/AndroidGeeksYT/kawaii) for best experience.

## Supported Integrations

- Bufferline.nvim
- Cmp.nvim
- Codeactionmenu
- Nvim-dap
- Nvim-webdevicons
- Hop.nvim
- Vim-illuminate
- Lsp ( diagnostics )
- Nvim Navic
- LspSaga
- Mason.nvim
- Notify.nvim
- Nvim-tree
- Telescope.nvim
- Rainbow-delimiters.nvim
- Todo.nvim
- Nvim-treesitter
- Lsp Semantic tokens
- Trouble.nvim
- Whichkey.nvim
- git-conflict.nvim
- Orgmode
- diffview.nvim
- leap.nvim
- Edgy.nvim
- Grug-far.nvim
- Flash.nvim
- Blink.nvim
- Blink.pairs
